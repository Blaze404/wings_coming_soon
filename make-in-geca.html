MAKE IN GECA

What is MAKE IN GECA?

MAKE IN GECA is an event of WINGS2k20. This event is totally based on coding and web development. Participant at least required a basic and moderate knowledge of C language, C++ Language, Java Language or Python Language, HTML, JavaScript, CSS, PHP. It gives the student chance to test their language skills as well as it will be helpful for their campus placement. It is open for all streams.

  • One participant can only have single entry
  • All rounds will be taken as per below mentioned
  • Time pattern is fixed and each round is expected to follow schedule
  • Participants are expected to give their contact details correctly to inform them about the event and the result
  • First two rounds are mandatory for participant to participate to get into third round
  • The elimination takes place on the basis of participants performance in first two rounds
  • On the first day we conduct first two rounds of event and second day there will be third round of event

Rules.

Round 1

  • We will provide you input and output, you have to write a logic in any two different languages.
  • Languages: C, C++, JAVA, PYTHON.
  • Duration: 60 Minutes or 30 Minutes.
  • The more correct logics you find the more points you will get.

Round 2

  • We will provide output on pages. Participant need to write the input within given specific time.
  • You can choose and write a code in any one language. (C,C++,JAVA,PYTHON)
  • Duration: 60 Minutes

Round 3

  • Based on any given theme participant have to design web page. (Template or minimum required source will be provided to students).
  • Duration: 120 Minutes
